# proj_portfolio

🌐 Aryan's Portfolio Website

A modern, interactive personal portfolio website built using HTML, CSS, and JavaScript, featuring smooth animations, custom cursor effects, and a responsive design.

---

🚀 Features

✨ Interactive UI/UX

- Custom animated cursor
- Smooth scrolling navigation
- Page transition animations

🎨 Modern Design

- Dark theme with highlight accents
- Handwriting SVG intro animation
- Responsive layout for mobile & desktop

🧑‍💻 Sections Included

- Landing page with animated intro
- About Me section with hover effects
- Latest Work (project slider)
- Contact form
- Social media integration

📱 Responsive Design

- Optimized for mobile, tablet, and desktop
- Custom behavior for smaller screens

---

🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (Vanilla)
- Google Fonts (Poppins, Montserrat, Caveat)
